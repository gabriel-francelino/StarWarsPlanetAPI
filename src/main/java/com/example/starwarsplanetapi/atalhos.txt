Ctrl + D - Duplica a linha

Ctrl + Alt + setas direita e esquerda -  Atera entre as classes abertas

Ctrl + Alt + L - Identa o código

Ctrl + Alt + O - Remove imports que não estão sendo utilizados

Ctrl + B - Mostra onde a váriavel foi criada

Alt + F7 - Indica todos os lugares onde a variável está sendo usada

Ctrl + Alt + M -  Extrai método automaticamente

Shift + F6 - Renomeia

Ctrl + Shift + N - Busca arquivos
